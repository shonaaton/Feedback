Webhook Setup

The portal is intentionally built around n8n webhooks and a thin Apps Script layer, closer to your older task manager pattern.

n8n should do the smart orchestration here: fetch real months, load tasks, normalize old approval data, fetch and correct Lichess activity, notify mentors, and send reminders. The frontend should mainly select, review, and submit.
