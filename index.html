What this portal now handles

One-click Lichess data

Rating, games, wins, draws, losses, rating change, puzzle activity, and best result are fetched through the backend.

Protected system fields

Month, student identity, coach identity, and batch information should come from backend data, not manual typing.

Mentor editing before approval

Mentors can update coach-written fields, approve, or return the report for changes.

Monthly automation

Task launch on the 26th, coach reminders, mentor alerts, and parent PDF delivery all live in the backend package.

n8n-friendly backend

n8n can be used to fetch, normalize, and repair legacy data so the portal stays clean and coaches avoid wrong inputs.
